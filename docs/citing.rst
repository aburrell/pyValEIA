Citation Guidelines
===================

When publishing work that uses pyValEIA, please cite the package and
any package it depends on that plays an important role in your analysis.
Specifying which version of pyValEIA used will also improve the reproducibility
of your presented results.

pyValEIA
--------

The most recent citation can be found at Zenodo. The examples here is from the
first public release.  A journal article describing this method is forthcoming.

* Cardenas-O'Toole, A., et al. (2025).
  pyValEIA: Alpha Release (Version 0.0.1).

.. code-block:: latex
   
    @Misc{pyValEIA,
          author = {Cardenas-O'Toole, A. and Burrell, A. G. and Hickey, D},
	  title  = {pyValEIA: Alpha Release},
  	  year   = {2025},
	  type   = {Software},
	  version = {v0.1.0},
	  }
